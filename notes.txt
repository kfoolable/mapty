// 1. User Stories
// - Description of the applications functionality from the user's perspective
// - Common format: As a [type of user], I want [an action] so that [a benefit].
// [type of user] Who? Example: user, admin, etc.
// [an action] What?
// [a benefit] Why?
// Example:
// 1. As a user, I want to log my running workout with location, distance, time, pace and steps/minute, so I can keep a log of my running.
// 2. As a user, I want to log my cycling workout with location, distance, time, speed and elevation gain, so I can keep a log of my cycling.
// 3. As a user, I want to see all my workouts at a glance, so I can easily track my progress overtime.
// 4. As a user, I want  to also see my workouts on a map, so I can easily check where I work out the most.
// 5. As a user, I want to see all my workouts when I leave the app and come back later, so that I can keep using their app over time.
